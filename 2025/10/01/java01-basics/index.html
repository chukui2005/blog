一、Java是什么

Java是一种面向对象的编程语言，由Sun公司（现属Oracle）于1995年发布。核心特点是 一次编写，到处运行——同一份代码可以在Windows、Linux、macOS等不同操作系统上运行，不需要修改。

这个跨平台能力靠的是 JVM（Java虚拟机）：编译后的Java代码不直接交给操作系统，而是交给JVM去解释执行。每个平台有对应版本的JVM，代码不用改。

二、搭建开发环境

JDK / JRE / JVM 三者的关系

    graph TD
    JDK[JDK 开发工具包] --> JRE[JRE 运行时环境]
    JDK --> JAVAC[编译器 javac]
    JDK --> JAR[打包工具 jar]
    JDK --> JDB[调试器 jdb / jstack]
    JRE --> JVM[JVM 虚拟机]
    JRE --> LIB[核心类库 rt.jar]
  • JVM：Java虚拟机，负责把字节码翻译成机器码执行。不同操作系统有不同版本的JVM，但对外接口一致。
  • JRE：Java运行时环境，包含JVM和Java核心类库。如果你只想运行Java程序，装JRE就够了。
  • JDK：Java开发工具包，包含JRE和开发工具（编译器javac、调试器jdb等）。如果要写Java代码，需要装JDK。

实际开发中直接装JDK就行，它已经包含了JRE。

四、基本数据类型

Java是强类型语言——每个变量必须先声明类型，然后才能使用。不像Python或JavaScript那样可以随便改变类型。

Java有 8种基本数据类型（primitive types），分为三类：数值型、字符型、布尔型。

数值型——整型（4种）

整数类型用来存储没有小数部分的数字，根据取值范围不同分为4种：

类型 占用字节 取值范围 默认值
byte 1 -128 ~ 127（2^7） 0
short 2 -32768 ~ 32767（2^15） 0
int 4 -2147483648 ~ 2147483647（2^31） 0
long 8 -2^63 ~ 2^63-1 0L

为什么long后面要加L？ 因为Java默认整数字面量是int类型。不加L的话，你写 10000000000 会超出int的范围，编译报错。

使用建议：日常开发中绝大部分场景用int就够了，涉及时间戳/文件大小等可能超int范围的用long。

数值型——浮点型（2种）

浮点型用来存储有小数部分的数字。

类型 占用字节 精度 默认值
float 4 约7位有效数字 0.0f
double 8 约15位有效数字 0.0d

为什么float要加f？ 因为带小数的数字默认是double类型。double占8字节，float占4字节，大转小需要显式声明。

类型转换

不同类型的数据一起运算时，需要类型转换。

自动类型转换（隐式转换）：小范围类型自动转成大范围类型。

转换方向：byte → short → int → long → float → double
（注意：char可以转int，但不参与byte→short这条线）

强制类型转换：大范围转小范围，需要手动加类型转换符。

表达式的自动类型提升：

  • byte、short、char 参与运算时，会先自动提升为int
  • 如果表达式中有long，整个提升为long
  • 如果表达式中有float，整个提升为float
  • 如果表达式中有double，整个提升为double

四、循环结构——for

for循环用于已知循环次数的场景。

执行流程分解：

  1. 执行 int i = 0（只执行一次）
  2. 判断 i < 5，为true则进入循环体，否则退出
  3. 执行循环体 System.out.println(i)
  4. 执行 i++
  5. 回到第2步

增强for循环（for-each）：遍历数组和集合时更简洁。

五、循环结构——while

while用于不知道循环次数，但知道循环条件的场景。

六、循环结构——do-while

do-while先执行一次循环体，再判断条件，所以循环体至少执行一次。

三种循环的选择：

  • 知道循环次数 → for
  • 不知道次数但知道条件 → while
  • 至少要执行一次 → do-while

Q：Java的基本类型有哪几种？分别占多少字节？

类型 字节
byte 1
short / char 2
int / float 4
long / double 8
boolean 未明确定义，JVM实现相关（通常占1个字节）

Q：3 * 0.1 在Java中等于多少？

Q：int 和 Integer 有什么区别？

int是基本类型，Integer是int的包装类（引用类型）。后续的文章会详细讲解。

Q：++i 和 i++ 哪个效率高？

对于基本类型，两者效率几乎没差别。在C++中前置++效率稍高（少了临时对象），但Java中无需考虑这个。

Q：Java中有goto吗？

goto 是Java的保留字，但Java语言中没有使用goto。Java用带标签的break/continue来实现类似功能。

Q：switch 支持的数据类型有哪些？

byte、short、char、int、枚举、String（JDK 7+）。
