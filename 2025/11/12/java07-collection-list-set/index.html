对比 Comparable Comparator

常见使用 TreeSet、Collections.sort() 临时排序需求

常见易错点

  1. 迭代时不能用集合的 remove：必须用迭代器的 remove，否则 ConcurrentModificationException
  2. TreeSet/HashSet 的 hashCode 和 equals 要求一致：否则元素去重逻辑异常
  3. TreeSet 不能存 null：红黑树比较时会 NPE
  4. ArrayList 的 subList 返回的是视图：对 subList 的修改会反映到原列表
  5. Set 不保证顺序（HashSet）或保证插入顺序（LinkedHashSet）或排序（TreeSet）
