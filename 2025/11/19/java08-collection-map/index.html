常见易错点

  1. HashMap 用可变对象作 key：如果修改了 key 的内部状态导致 hashCode 变化，将无法找到该 entry
  2. ConcurrentHashMap 不允许 null：与 HashMap 不同
  3. TreeMap/TreeSet 的 key 必须实现 Comparable 或提供 Comparator：否则 ClassCastException
  4. Collections.singletonList 返回的是不可变 List：不能 add/remove
  5. PriorityQueue 的迭代器不保证顺序：只有 poll/peek 才按优先级返回
